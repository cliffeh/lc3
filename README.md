# lc3
an LC-3 VM

## What is this?
[Write your Own Virtual Machine](https://www.jmeiners.com/lc3-vm/) inspired me to try my hand at writing my own LC-3 VM.

## Tips
For dumping object files in a diffable way (one instruction per line): `od -x -w2`
